Bookiny

December 2019

Visit app

Project aimed to build a web application that would allow people to digitalize your bookshelf.

User can add the books they own making an inventory (or list) of all the books you own. Once store in the app, they can be search quickly.

Users can become friends and see which books their friends own.

If you are looking for a new book to buy, before doing that you could check if any of your friends already owns the book you would like to buy, and borrow it.

Same case if a friend is looking for a book you own. You could lend it to him.

Features:

  • Search quickly the books you own
  • Add new book automatically with just the book's ISBN-10 or ISBN-13 code
  • Add new book manually.

tools: Python
